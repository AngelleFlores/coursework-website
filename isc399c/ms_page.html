Assignment #2 — Minesweeper App Version 6

This assignment is a version of the Minesweeper computer game. This version of the game includes sounds and images.


This is how the startup screen looks like. A sound will
play at the beginning and an image will be displayed.
One would enter values for the number of rows, columns,
and mines (in that order) in the EditText items. Then, the
"Build Grid" button would be pressed to create your own
customized Minesweeper game.
  • Used MediaPlayer to play sound


This is the screen that would be viewed after clicking
the "Build Grid" button.
  • Used ScrollView


This is an example of what you would see if you pressed
all the TextView tiles. This is to demonstrate the use of
images and sounds. If you pressed on a tile that contained
a mine, you would hear an explosion sound and see an image
of a mine. If you press any other tile, you would hear a
clicking sound and see the number representing the number
of mines surrounding that particular tile.
  • Used BitmapFactory and Bitmap to create mine images
  • Used SoundPool to play the sounds


This is an example of what you would see if you won the
game. You would not see any mine images and a "tada" sound
would play to declare that you are the winner.
  • Used SoundPool to play sound
