Assignment #3 — Swiping App Version 1

Notes

  • The "Drop Shape" button adds a moving block to the canvas.
  • The EditText views allow you to decide the speed of the moving block in x and y directions.
  • The blocks will bounce off all four sides of the canvas.
  • Each block that is added to the canvas will be given a random color.
  • The "Remove All Shapes" button clears the canvas.
  • Tapping a block once stops it from moving.
  • Tapping a block twice removes it from the canvas.
  • Pressing a block and swiping in a certain direction flings it across the canvas.
